# scratchpad

Edit the session scratchpad using multiple operations in a single call. Operations can be str_replace, insert, or overwrite commands and are applied in order. The scratchpad is working memory that persists when conversation history is compacted or summarized.

When using **str_replace**:
• Prefer a very short, *exact* substring that uniquely appears in the scratchpad.  
• Avoid pasting long blocks of text—smaller, specific targets make replacement more reliable.  

The scratchpad content returned by this tool is **always annotated with line numbers** (e.g. `12: …`). Use those numbers to reason about where to insert or replace text.

If no scratchpad exists yet, initialize with overwrite using this structure:
### Goal
*Overall objective*

### Plan  
*High-level milestones*
- [ ]

### Current Tasks  
*Steps for current milestone*
- [ ]

### Knowledge
*Facts you've discovered*

### Failed Approaches
*What didn't work and why*

### Open Questions
*Need answers to proceed*

### Notes
*Scratch space*
